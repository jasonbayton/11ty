**Features & improvements**

- Studio now shows real Google Play app icons in the device preview and app palette when they are available.
- Studio now starts with an empty app grid rather than pre-placed sample apps.
- Entrance and motion animations now play on chapter changes rather than replaying on every input, click or selection.

**Fixes**

- Fixed narrow-width build overlays that could overflow because of long update codes or package names.
- Fixed glyph icon centring inside tile backgrounds.
- Fixed Studio review edit affordances and several reference-prototype parity issues around admin escape fields, transient state, preview cleanup and build progress logging.
