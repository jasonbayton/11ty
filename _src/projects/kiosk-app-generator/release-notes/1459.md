**Features & improvements**

- Studio assets are now served with `Cache-Control: no-cache` so browsers revalidate the un-hashed ES modules after deployment.

**Fixes**

- Fixed Studio navigation and review gating so they refresh correctly while typing launcher names or emptying the grid.
- Fixed the grid-size dropdown so custom remains visible when a non-preset layout is typed.
- Fixed update-code handling so lowercase hex codes are not force-uppercased.
- Fixed imported `kiosk_config.json` persistence immediately after import.
- Fixed the Studio docs full-reference deep link so it scrolls accurately to the target section.
