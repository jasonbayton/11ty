**Features & improvements**

- Added optional admin escape: a hidden, password-gated admin app drawer opened by long-pressing the settings slot or shield icon.
- Added build-form and managed-configuration support for admin escape, including password rotation and a configurable list of admin apps.
- Admin drawer apps can be kept off the main kiosk grid while still using normal Android launch and lock-task policy enforcement.
