**Features & improvements**

- Initial stable CONTACTS APP GENERATOR release.
- Generates signed Android Managed Contacts apps from VCF, CSV, or JSON sources.
- Supports built-in contact files, URL-backed sources, managed configuration, Studio and classic build surfaces, signing modes, source export, config export, and update-code reuse.
- Raises embedded source-file support to 32 MiB to match the generated app's runtime source ceiling.
