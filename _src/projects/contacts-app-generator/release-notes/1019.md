**Features & improvements**

- Added `branding_accent` as a flat managed-configuration key for generated CAG apps.
- Deployed apps can now be re-tinted by EMM managed configuration without a rebuild.
- Runtime accent resolution now uses the effective managed configuration first, then falls back to the baked build accent.
- Validated the key through Android's managed-configuration path on a generated app.

**Fixes**

- Corrected CAG documentation to describe the accent as EMM-configurable.
